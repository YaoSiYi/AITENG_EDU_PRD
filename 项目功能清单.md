# 艾腾教育 - 软件测试全栈工程师培训系统

## 项目功能清单

**版本**: v0.1.0  
**更新日期**: 2026-04-24  
**技术栈**: Django + Vue 3 + Uni-app

---

## 一、项目定位

专为软件测试工程师培训设计的在线学习平台，提供从零基础到高级的完整学习路径，包含10个阶段的系统化课程、实战项目和AI大模型测试等前沿内容。

---

## 二、课程体系

| 级别 | 阶段 | 内容 |
|------|------|------|
| **初级** | 第1-3阶段 | 软件测试基础、功能与非功能测试、Linux和MySQL |
| **中级** | 第4-7阶段 | Python、UI自动化、接口测试、性能测试 |
| **高级+进阶** | 第8-10阶段 | 接口自动化、项目实战（金融/微商/银行）、AI大模型与Agent智能体 |

## 四、安全特性

| 特性 | 说明 |
|------|------|
| JWT认证 | 所有API需要登录验证 |
| 路由守卫 | 前端页面需要登录 |
| CSV公式注入防护 | 导入时自动检测 |
| XSS脚本检测 | 防止恶意脚本注入 |
| 文件大小限制 | 2MB限制 |
| 表头白名单校验 | 导入时验证表头 |
| 审计追踪 | 记录创建人、修改人、时间戳 |

## 五、技术架构

```
┌─────────────────────────────────────────────────────────┐
│  用户端 (Web / App / 小程序)                              │
│  - Vue 3 Web (学生/老师/游客切换)                         │
│  - Uni-app (微信小程序 + App + H5)                       │
└────────────────┬────────────────────────────────────────┘
                 │ REST API + WebSocket
                 ↓
┌─────────────────────────────────────────────────────────┐
│  后端服务 (Django + DRF)                                 │
│  - Django REST Framework (API)                          │
│  - JWT 认证 (djangorestframework-simplejwt)             │
│  - Celery (异步任务 + 定时备份)                          │
└────────────────┬────────────────────────────────────────┘
                 │
    ┌────────────┼────────────┐
    ↓            ↓            ↓
┌────────┐  ┌────────┐  ┌────────┐
│ MySQL  │  │ Redis  │  │ Nginx  │
│  8.x   │  │  7.x   │  │ (反代) │
└────────┘  └────────┘  └────────┘
```
