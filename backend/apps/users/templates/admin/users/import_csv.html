📋 导入说明

  • 请使用 CSV 格式文件（UTF-8 编码）
  • 必填字段：username（用户名）、password（密码）
  • 可选字段：nickname（昵称）、phone（手机号）、qq（QQ号）、hometown（籍贯）、password_hint（密码提示）、period（期数）、study_status（学习状态）、role（角色）、is_active（是否激活）
  • 学习状态可选值：studying（在读）、graduated（毕业），默认为 studying
  • 角色可选值：student（学生）、teacher（老师）、admin（管理员），默认为 student
  • 是否激活可选值：True、False，默认为 True
  • 如果用户名已存在，该行将被跳过

  • 建议先小批量测试导入，确认无误后再批量导入
