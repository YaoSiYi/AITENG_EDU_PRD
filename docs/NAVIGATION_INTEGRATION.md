## 🚀 快速开始

### 1. 启动后端服务
```bash
cd backend
source venv/bin/activate
python manage.py runserver 8000
```

### 2. 启动前端服务
```bash
cd frontend-web
npm run dev
```

### 3. 访问系统
1. 打开浏览器访问 `http://localhost:5173`
2. 登录系统（如果还没有账号，先注册）
3. 在顶部导航栏可以看到：
   - **测试用例** 菜单项
   - **数据看板** 菜单项

### 4. 使用测试用例管理
1. 点击「测试用例」进入管理页面
2. 点击「新建」创建测试用例
3. 使用筛选条件查询
4. 点击「导出」下载CSV
5. 点击「导入」批量导入

### 5. 查看数据看板
1. 点击「数据看板」进入可视化页面
2. 查看统计卡片
3. 查看图表分析

## 📝 注意事项

### 1. 登录状态
- 测试用例和数据看板功能需要登录才能访问
- 如果未登录，导航菜单中不会显示这两个选项
- 直接访问URL会被路由守卫拦截并重定向到登录页

### 2. 权限验证
- 路由配置中已添加 `meta: { requiresAuth: true }`
- 确保用户已登录才能访问

### 3. 数据依赖
- 测试用例管理需要后端API正常运行
- Dashboard图表需要从API获取数据
- 确保后端服务已启动（`python manage.py runserver 8000`）
