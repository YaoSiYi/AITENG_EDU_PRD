## 新增功能

### 1. 确认密码框

**位置**: 注册页面

**功能特性**:
- ✅ 输入框自动识别密码一致性
- ✅ 实时验证两次密码是否一致
- ✅ 不一致时显示错误提示
- ✅ 支持显示/隐藏密码

**验证规则**:
```javascript
const validateConfirmPassword = (rule, value, callback) => {
  if (value === '') {
    callback(new Error('请再次输入密码'))
  } else if (value !== form.password) {
    callback(new Error('两次输入的密码不一致'))
  } else {
    callback()
  }
}
```

### 2. 密码提示字段

**位置**: 注册页面、个人中心设置页、管理后台

**功能特性**:
- ✅ 选填字段
- ✅ 用于找回密码的提示信息
- ✅ 最多 200 个字符
- ✅ 支持中英文

**数据库字段**:
```python
password_hint = models.CharField(
    '密码提示', 
    max_length=200, 
    blank=True, 
    default='', 
    help_text='用于找回密码的提示信息'
)
```

### 3. 籍贯字段

**位置**: 注册页面、个人中心设置页、管理后台

**功能特性**:
- ✅ 格式验证：xx省xx市
- ✅ 实时格式检查
- ✅ 格式错误时显示提示
- ✅ 支持中文省市名称

**格式验证**:
```javascript
const validateHometown = (rule, value, callback) => {
  if (!value) {
    callback()
    return
  }
  const pattern = /^.+省.+市$/
  if (!pattern.test(value)) {
    callback(new Error('籍贯格式应为：xx省xx市'))
  } else {
    callback()
  }
}
```

**示例**:
- ✅ 正确：北京省北京市
- ✅ 正确：广东省广州市
- ✅ 正确：四川省成都市
- ❌ 错误：北京市
- ❌ 错误：广东广州
- ❌ 错误：Beijing

## API 接口

### 注册接口

**端点**: `POST /api/users/register/`

**请求体**:
```json
{
  "username": "testuser",
  "nickname": "测试用户",
  "phone": "13800138000",
  "hometown": "北京省北京市",
  "password": "password123",
  "confirm_password": "password123",
  "password_hint": "我的生日"
}
```

**成功响应** (201):
```json
{
  "message": "注册成功",
  "user": {
    "id": 1,
    "username": "testuser",
    "nickname": "测试用户",
    "phone": "13800138000",
    "hometown": "北京省北京市",
    "password_hint": "我的生日",
    "role": "student",
    "created_at": "2026-04-24T10:00:00Z"
  }
}
```

**错误响应** (400):
```json
{
  "confirm_password": "两次密码不一致"
}
```

或

```json
{
  "username": ["该用户名已被使用"]
}
```

## 数据库变更

### 新增字段

```sql
ALTER TABLE users ADD COLUMN password_hint VARCHAR(200) DEFAULT '';
ALTER TABLE users MODIFY COLUMN hometown VARCHAR(100) DEFAULT '' COMMENT '格式：xx省xx市';
```

### 字段说明

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|--------|------|------|------|--------|------|
| password_hint | VARCHAR | 200 | 否 | '' | 密码提示 |
| hometown | VARCHAR | 100 | 否 | '' | 籍贯（xx省xx市） |

## 注意事项

### 1. 籍贯格式

- ✅ 必须包含"省"和"市"
- ✅ 支持直辖市：北京省北京市
- ✅ 支持自治区：新疆省乌鲁木齐市
- ❌ 不支持：北京市、广州市

### 2. 密码提示

- 不要使用密码本身作为提示
- 建议使用问题形式：如"我的生日"、"我的宠物名字"
- 不要包含敏感信息

### 3. 确认密码

- 实时验证，失去焦点时触发
- 必须与密码完全一致
- 区分大小写

### 4. 数据迁移

- 已有用户的 `password_hint` 默认为空
- 已有用户的 `hometown` 保持原值
- 不影响现有数据
