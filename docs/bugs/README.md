## Bug 列表

### BUG-001: Python 3.14 + Django 5.x 兼容性问题
- **日期**: 2026-04-22
- **严重程度**: 🔴 High
- **状态**: ✅ 已解决
- **文档**: [BUG-001-python314-django-compatibility.md](./BUG-001-python314-django-compatibility.md)
- **影响**: 后端服务无法启动
- **解决方案**: 升级 Django 到 6.0.4，升级 mysqlclient 到 2.2.8

### BUG-002: 前端登录接口 404（缺少 /api 前缀）
- **日期**: 2026-04-22
- **严重程度**: 🔴 High
- **状态**: ✅ 已解决
- **文档**: [BUG-002-frontend-login-404.md](./BUG-002-frontend-login-404.md)
- **影响**: 用户无法登录，测试用例管理页面显示为空
- **解决方案**: 修复 `user.js` 和 `api/index.js` 中的 API 路径，添加 `/api` 前缀

### BUG-003: 数据看板显示静态数据
- **日期**: 2026-04-22
- **严重程度**: 🟡 Medium
- **状态**: ✅ 已解决
- **文档**: [BUG-003-dashboard-static-data.md](./BUG-003-dashboard-static-data.md)
- **影响**: 数据看板图表显示硬编码的模拟数据，未实时同步测试用例统计
- **解决方案**: 修改 `Dashboard.vue`，实时获取并统计测试用例数据

### BUG-004: 测试用例与数据看板页面缺少返回首页路由入口
- **日期**: 2026-04-22
- **严重程度**: 🟡 Medium
- **状态**: ✅ 已解决
- **文档**: [BUG-004-missing-home-navigation.md](./BUG-004-missing-home-navigation.md)
- **影响**: `/testcases`、`/dashboard` 页面没有返回首页入口
- **解决方案**: 为两个页面新增面包屑导航（首页/页面名）

### BUG-005: 个人中心编辑资料按钮无效
- **日期**: 2026-04-23
- **严重程度**: 🟡 Medium
- **状态**: ✅ 已解决
- **文档**: [BUG-005-profile-edit-button-not-working.md](./BUG-005-profile-edit-button-not-working.md)
- **影响**: `/profile` 页面点击"编辑资料"按钮无反应
- **解决方案**: 新增编辑弹窗组件和保存逻辑

### BUG-006: 测试用例状态缺少“已关闭”
- **日期**: 2026-04-23
- **严重程度**: 🟢 Low
- **状态**: ✅ 已解决
- **文档**: [BUG-006-testcase-status-closed.md](./BUG-006-testcase-status-closed.md)
- **影响**: 测试用例状态流转不完整
- **解决方案**: 后端状态枚举新增 `closed` 并同步前端所有状态入口

### BUG-007: 测试用例管理页为空
- **日期**: 2026-04-23
- **严重程度**: 🔴 High
- **状态**: ✅ 已解决
- **文档**: [BUG-007-testcase-page-empty.md](../BUG-007-testcase-page-empty.md)
- **影响**: 测试用例管理页面无法显示数据
- **解决方案**: 修复端口冲突、补充缺失函数定义（`getStageType`、`getStageLabel`）、补充缺失导入（`getAssignableUsers`、图标组件）

## 记录规范

每个 Bug 文档应包含：

1. **基本信息**
   - 日期
   - 严重程度（🔴 High / 🟡 Medium / 🟢 Low）
   - 状态（✅ 已解决 / ⏳ 进行中 / 🔴 待解决）
   - 影响范围

2. **问题现象**
   - 错误信息
   - 复现步骤
   - 截图（如有）

3. **根因分析**
   - 问题原因
   - 技术细节

4. **修复内容**
   - 修改的文件
   - 具体改动
   - 代码示例

5. **验证结果**
   - 测试步骤
   - 验证结果

6. **预防措施**
   - 如何避免类似问题
   - 改进建议
