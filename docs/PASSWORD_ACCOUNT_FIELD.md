# 个人中心密码修改 - 账号字段优化

## 更新说明

在个人中心设置页的密码修改模块中，新增了账号字段显示，位于原始密码输入框上方。

---

## 修改内容

### 1. 新增账号字段

**位置**: 个人中心 → 设置 → 修改密码模块

**字段特性**:
- ✅ 显示当前登录用户的用户名
- ✅ 置灰状态（disabled）
- ✅ 不可编辑
- ✅ 自动获取用户信息

### 2. 数据来源

账号字段的值优先从以下来源获取：
1. **userStore.user.username** - 用户 Store 中的用户名（优先）
2. **settingsForm.username** - 设置表单中的用户名（备用）

### 3. 代码实现

```vue
<el-form-item label="账号">
  <el-input
    :value="userStore.user?.username || settingsForm.username"
    disabled
    placeholder="当前登录账号"
  />
</el-form-item>
```

**关键属性**:
- `:value` - 绑定用户名值（使用可选链操作符 `?.` 防止空值错误）
- `disabled` - 设置为禁用状态（置灰不可编辑）
- `placeholder` - 占位符文本

## 技术细节

### 1. 数据绑定

使用 `:value` 而不是 `v-model`：
- `:value` 单向绑定，适合只读字段
- `v-model` 双向绑定，适合可编辑字段

### 2. 可选链操作符

使用 `userStore.user?.username`：
- 防止 `userStore.user` 为 `null` 或 `undefined` 时报错
- 如果 `user` 不存在，返回 `undefined`，然后使用备用值

### 3. 备用值

使用 `||` 运算符提供备用值：
- 如果 `userStore.user?.username` 为空，使用 `settingsForm.username`
- 确保字段始终有值显示

## 样式说明

### 置灰效果

Element Plus 的 `disabled` 属性会自动应用以下样式：
- 背景色：浅灰色 `#f5f7fa`
- 文字颜色：灰色 `#c0c4cc`
- 光标：`not-allowed`（禁止图标）
- 边框：灰色

| 新密码 | normal | 白色 | ✅ 是 |

## 注意事项

### 1. 数据安全

- 账号字段只显示用户名，不显示敏感信息
- 密码字段仍然使用密码输入框（`type="password"`）

### 2. 兼容性

- 使用可选链操作符 `?.` 需要现代浏览器支持
- 如果需要兼容旧浏览器，可以使用 `userStore.user && userStore.user.username`

### 3. 用户体验

- 置灰状态明确表示不可编辑
- 如果用户尝试点击，不会有任何反应
- 光标会显示为禁止图标
