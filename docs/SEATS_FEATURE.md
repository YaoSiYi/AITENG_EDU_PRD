# 查看座次功能文档

## 更新日期：2026-04-24

---

## 🎯 功能概述

为主页新增"查看座次"模块，展示教室座位布局，支持两种布局切换，可视化显示座位占用情况。

---

## ✨ 新增功能

### 1. 导航栏入口

在顶部导航栏"个人中心"左侧新增"查看座次"链接。

**位置**：
```
题库 | 活动 | 统计 | 查看座次 | 错题本 | 测试用例 | 数据看板 | 个人中心
                      ↑ 新增
```

### 2. 主页按钮入口

在主页"查看活动"按钮右侧新增"查看座次"按钮。

**样式**：
- 颜色：科技绿 (#10b981)
- 大小：与其他按钮一致
- 位置：查看活动按钮右侧

### 3. 座位布局页面

#### 布局类型

**布局一：8列6排（48座）**
- 横向：8列
- 左侧：4个座位
- 中间：过道
- 右侧：4个座位
- 纵向：6排
- 总座位：48个

**布局二：10列8排（80座）**
- 横向：10列
- 左侧：5个座位
- 中间：过道
- 右侧：5个座位
- 纵向：8排
- 总座位：80个

#### 座位状态

**空座位**：
- 颜色：白色（半透明）
- 边框：白色边框
- 状态：可点击

**女生座位**：
- 颜色：粉色渐变 (#ec4899 → #f472b6)
- 显示：姓名
- 状态：已占座

**男生座位**：
- 颜色：湖蓝色渐变 (#06b6d4 → #22d3ee)
- 显示：姓名
- 状态：已占座

## 🚀 使用步骤

### 方式一：通过导航栏访问

1. 访问主页：http://localhost:3000/
2. 点击顶部导航栏的"查看座次"链接
3. 进入座位布局页面

### 方式二：通过主页按钮访问

1. 访问主页：http://localhost:3000/
2. 在主页中间区域找到"查看座次"按钮（科技绿色）
3. 点击按钮进入座位布局页面

### 切换布局

1. 在座位布局页面顶部
2. 点击"8列6排 (48座)"或"10列8排 (80座)"按钮
3. 布局会立即切换

### 查看座位信息

1. 空座位：白色显示，显示座位编号
2. 已占座：显示学生姓名
3. 女生：粉色背景
4. 男生：湖蓝色背景
