## 🚀 快速启动

### 1. 安装依赖
```bash
cd frontend-uniapp
npm install
```

### 2. 配置API地址
修改 `utils/request.js` 中的 `BASE_URL`：
```javascript
const BASE_URL = 'http://localhost:8000/api'
```

### 3. 运行项目

**微信小程序**:
```bash
npm run dev:mp-weixin
```

**H5**:
```bash
npm run dev:h5
```

**App**:
```bash
npm run dev:app
```

### 1. 开发顺序建议
1. **第一步**: 准备静态资源（Logo、图标、轮播图）
2. **第二步**: 配置API地址并测试接口连通性
3. **第三步**: 完善题目详情页和活动详情页
4. **第四步**: 测试核心流程（登录→刷题→错题本）
5. **第五步**: 优化用户体验和界面细节

### 2. 注意事项
